The EasyA system is a program that students can use to display graphs
about past instructor grade data at the University of Oregon.

Authors: Meagan Beckstrand, Eliza Black, Etienne Casal-Jouaux, Angel Soto, Connie Williamson
Date Created: 2/4/2024
This was created as an assignment for CS 422 as Project 1.
Setup procedures are described in Documentation/User_Documentation.pdf. 
Software Dependencies: Python 3 must be installed on the intended machine.

Subdirectories:
    Documentation: this contains all PDF files including user instructions and programmers documentation.
    Source Code: contains all code files that interact to form the EasyA program. Also contains the data files (both .js and .csv) for past instructor data.
